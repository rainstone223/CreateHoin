# 호인 키우기 🐯

"아기고래쿠아" 스타일의 성장형 웹 게임입니다. 작은 호인 캐릭터가 다른 개체를 잡아먹으며 성장하여 최종 단계 "호인"에 도달하면 승리합니다.

## 🎮 게임 플레이

- **조작**: WASD 키 또는 화살표 키로 호인을 조작
- **목표**: 초록색 개체를 먹어 경험치 획득, 최종 레벨 "호인" 도달
- **주의**: 빨간색 개체에 닿으면 게임 오버!

## 📊 레벨 시스템

총 6단계의 레벨을 거쳐 성장합니다:

```
사원 (12px) → 대리 (16px) → 과장 (21px) → 부장 (28px) → 사장 (37px) → 호인 (50px) ✨
```

## 🚀 실행 방법

1. 저장소 클론
```bash
git clone https://github.com/rainstone223/CreateHoin.git
cd CreateHoin
```

2. `index.html` 파일을 브라우저에서 열기
   - 더블클릭하거나
   - 브라우저 주소창에 파일 경로 입력

## 🎯 게임 메커니즘

### 개체 시스템
- **먹이 (초록색)**: 5-10마리, 플레이어를 피해 도망
- **포식자 (빨간색)**: 4-6마리, 플레이어를 추적
- **플레이어 (파란색)**: 마우스를 따라 이동

### 성장 시스템
- 먹이를 먹으면 **10 XP** 획득
- 일정 경험치 도달 시 **레벨업**
- 레벨이 오를수록 크기 증가 (약 20%)

## ⌨️ 조작 키

### 게임 조작
- `W` 또는 `↑`: 위로 이동
- `A` 또는 `←`: 왼쪽으로 이동
- `S` 또는 `↓`: 아래로 이동
- `D` 또는 `→`: 오른쪽으로 이동

### 디버그 키
- `L`: 즉시 레벨업
- `G`: 무적 모드 토글
- `R`: 게임 리셋

## 🛠️ 기술 스택

- **HTML5 Canvas**: 게임 렌더링
- **JavaScript (ES6)**: 게임 로직
- **CSS3**: 스타일링

## 📁 프로젝트 구조

```
CreateHoin/
├── index.html           # 메인 HTML
├── css/
│   └── styles.css      # 스타일시트
├── js/
│   ├── config.js       # 게임 설정
│   ├── main.js         # 게임 루프
│   ├── Player.js       # 플레이어 클래스
│   ├── Entity.js       # 개체 베이스 클래스
│   ├── Prey.js         # 먹이 클래스
│   ├── Predator.js     # 포식자 클래스
│   ├── EntityManager.js    # 개체 관리
│   ├── CollisionDetector.js # 충돌 감지
│   ├── LevelSystem.js  # 레벨 시스템
│   ├── InputHandler.js # 입력 처리
│   ├── Renderer.js     # 렌더링
│   └── UI.js          # UI 시스템
└── assets/
    └── images/        # 이미지 리소스
```
